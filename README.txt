GYMBRUT - NetBeans Java Swing Starter Project

Isi ZIP:
- src/ : source code Java Swing
- gymbrut.sql : schema database MySQL + dummy data
- nbproject/ : file konfigurasi dasar NetBeans Ant project
- assets/ : folder asset kosong
- reports/ : folder report kosong

Cara pakai di Apache NetBeans:
1. Extract ZIP.
2. Buka NetBeans.
3. File > Open Project > pilih folder Gymbrut.
4. Jalankan file gymbrut.sql di MySQL.
5. Tambahkan library MySQL Connector/J ke project.
6. Sesuaikan username/password DB di src/config/DBConnection.java.
7. Run Project.
